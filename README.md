# Шпаргалка по работе с командами git
#### git clone: создает локальную копию (клон) удаленного репозитория на вашем компьютере.
#### git commit -m: создает коммит с кратким описанием изменений.
#### git status: отображает текущее состояние рабочей директории и области проиндексированных изменений (staging area).
#### git add .: индексировать все изменения (включая удаления).
#### git push: загружает локальные коммиты в удаленный репозиторий.
